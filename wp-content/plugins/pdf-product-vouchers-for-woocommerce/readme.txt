Create, manage, and send customized PDF vouchers for WooCommerce products.

== Description ==

PDF Product Vouchers for WooCommerce is a powerful plugin that allows you to create, manage, and sell PDF gift vouchers directly within your WooCommerce store. Customers can purchase, download, and redeem vouchers for products or services. With an intuitive interface, you can customize voucher designs, add discount codes, set expiration dates, and include other relevant details. This is an ideal solution for stores selling gift cards, discount coupons, event tickets, or prepaid services.	

== Installation ==
**Normal installation**

1. Download the woocommerce.zip file to your computer.
1. Unzip the file.
1. Upload the `plugin.zip` directory to your `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
Document include in plugin
