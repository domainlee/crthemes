The PDF For WooCommerce add-on is an additional plugin that extends the functionality of WooCommerce by providing features for generating PDF files: Invoices, Packing Slips, Shipping Labels, or any PDFs. 

Once you finish designing your PDF you can configure it to be attached to any Woo-commerce email. So you can sort out the completed email so the customers can get it after purchasing something.

== PDF Invoices ==
The PDF Invoice Plugin for WooCommerce is a useful tool that automatically generates and manages invoices in PDF format for orders on a WooCommerce store.

== PDF Packing Slips ==
It allows store owners to generate detailed packing slips in PDF format, listing items included in a shipment along with essential order details such as product names, quantities, and SKU numbers.

== PDF Delivery Notes ==
The plugin facilitate the delivery process by generating concise documents that accompany shipments. These notes typically include key information such as customer details, order number, product names, and quantities, serving as a handover record between the seller and the delivery personnel or customer. 

== PDF Shipping Labels ==
The plugins simplifies the shipping process by enabling the creation of professional shipping labels directly from the store’s admin panel. These labels typically include critical details such as sender and recipient addresses, order number, and barcode for tracking. Key features include customizable label layouts to comply with carrier requirements, support for multiple shipping providers, and options to print labels individually or in bulk. Integrated with order data, this feature saves time, reduces manual errors, and streamlines fulfillment by providing ready-to-use labels for packages, enhancing overall shipping efficiency.

== PDF Cart ==
The plugin allows users to generate a PDF document summarizing the contents of their shopping cart. This typically includes details such as product names, quantities, prices, and totals. Key functionalities include customizable templates to reflect your store’s branding, options to download or email the PDF to customers, and integration with the checkout process. This feature enhances customer convenience by providing a portable, shareable cart summary, useful for quotes, order previews, or record-keeping before finalizing a purchase.

== PDF Product==
The plugin enables the generation of PDF documents for individual products or product catalogs. These PDFs typically include detailed information such as product names, descriptions, prices, images, and specifications. Key features include customizable layouts to align with your brand, options to download or share the PDF with customers, and bulk generation for multiple products. This functionality is ideal for creating product brochures, spec sheets, or downloadable resources, enhancing marketing efforts and providing customers with offline access to product details.


== PDF Vouchers ==
The plugin allows store owners to create and distribute gift vouchers or coupons in PDF format. These vouchers typically include details such as voucher code, value, expiration date, and recipient information. Key features include customizable designs to match your branding, options to email vouchers directly to customers or recipients, and support for printable formats. This feature enhances gifting options, simplifies voucher management, and provides a professional, tangible way for customers to share or redeem offers, boosting sales and engagement.

== Drag and Drop Elements ==
The plugin offers a web-based drag-and-drop PDF template editor that lets you design your unique templates and generate PDF documents from reusable templates without coding.

The PDF template editor includes a rich set of layouts and components to develop beautiful PDFs with powerful expressions, formatting, page breaks, and data-binding.
We support Table, QRCode, Barcode, Text Labels, Page Break, Divider, Image, Spacer, HTML content, and other elements.

== Installation ==
**Normal installation**

1. Download the woocommerce.zip file to your computer.
1. Unzip the file.
1. Upload the `plugin.zip` directory to your `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
